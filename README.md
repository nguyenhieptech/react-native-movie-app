## Setup

1. Clone the project:

```bash
git clone https://github.com/nguyenhieptech/react-native-movie-app
```

2. Install dependencies

```bash
cd react-native-movie-app
yarn install
```

3. Pod install for iOS

```bash
npx pod-install
```

4. Start Metro Bundler

```bash
yarn start
```

5. Open new terminals, build for iOS/Android

```bash
yarn ios
```

```bash
yarn android
```
